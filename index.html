Weather App

As a part of the SheCodes Plus final project. , I designed and coded a fully functioning weather app, designed and coded using HTML,CSS, Bootsrap and JavaScript. Integrating data from the OpenWeather API. Features include searching by location or using current location, popular cities forecast, a 5 day forecast, icon integration. Using dynamic background themes to correlate with time of day as well as user selection.
